# Análisis de Exportaciones de Cobre — Perú → América del Norte (Q1 2026)
 
Análisis estadístico descriptivo de 133 lotes de exportación de cobre refinado (partida arancelaria 7407100000) enviados desde Perú hacia Estados Unidos, Canadá y México durante el primer trimestre de 2026.
 
## Contenido
 
- Distribución de exportaciones por país de destino (tabla de doble entrada, gráfico de barras y circular)
- Distribución de frecuencias del valor US$ FOB (Regla de Sturges, histograma)
- Variabilidad del peso neto por país de destino (rango intercuartílico, diagrama de cajas)
## Herramientas
 
Excel avanzado (tablas dinámicas, distribución de frecuencias, diagramas de caja), muestreo estadístico sistemático y estratificado.
 
## Fuente de datos
 
ADEX Data Trade (2026) — Aduanas del Perú.

Trabajo final del curso Estadística Aplicada I. Desarrollado en el archivo `dashboard-cobre.pdf` corresponde a la sección de análisis y visualización del trabajo.
